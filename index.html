厚生労働省「健康づくりのための睡眠ガイド2023」では成人に6時間以上（推奨7時間以上）を推奨。睡眠6時間未満が続くと、認知機能・感情コントロール・記憶力が著しく低下します。まず睡眠時間の確保を優先してください。
`; } else if (issues.includes('work-over')) { alertEl.innerHTML = `

睡眠7時間以上・仕事10時間未満を確保できています。適切な時間設計です。
`; } }
